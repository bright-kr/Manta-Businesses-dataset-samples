이 **Manta Businesses Dataset Sample**에는 **1,547개 레코드**가 포함되어 있으며 **Bright Data API**를 사용하여 추출되었습니다.

## Data Points

| Field Name | Description |
|------------|-------------|
| `company_name` | 비즈니스 엔터티의 공식 명칭입니다. |
| `business_state` | 비즈니스가 등록되어 있거나 운영 중인 주(State)입니다. |
| `sic_code` | 비즈니스 유형을 나타내는 Standard Industrial Classification 코드입니다. |
| `phone_number` | 비즈니스의 연락처 전화번호입니다. |
| `website` | 비즈니스의 공식 웹사이트 URL입니다. |
| `business_categories` | 비즈니스 활동 및 산업을 정의하는 카테고리입니다. |
| `business_city` | 비즈니스가 위치한 도시입니다. |
| `business_country` | 비즈니스가 위치한 국가입니다. |
| `business_zip_code` | 비즈니스 위치의 우편번호입니다. |
| `closed_on` | 해당되는 경우 비즈니스가 폐업했는지 여부를 나타냅니다. |
| `opening_hours` | 비즈니스의 영업 시작 시간입니다. |
| `closing_hours` | 비즈니스의 영업 종료 시간입니다. |
| `latitude` | 비즈니스 위치의 지리적 위도 좌표입니다. |
| `longitude` | 비즈니스 위치의 지리적 경도 좌표입니다. |
| `location_type` | 비즈니스 위치 유형(예: 소매점, 사무실)입니다. |
| `year_established` | 비즈니스 설립 연도입니다. |
| `avg_opening_hour` | 비즈니스가 일반적으로 영업을 시작하는 평균 시간(시)입니다. |
| `avg_closing_hour` | 비즈니스가 일반적으로 영업을 종료하는 평균 시간(시)입니다. |
| `business_street_name` | 비즈니스 위치의 도로명 주소입니다. |
| `description` | 비즈니스 및 운영에 대한 간략한 개요입니다. |
| `services` | 비즈니스가 제공하는 서비스 목록입니다. |
| `links` | 비즈니스와 연관된 추가 링크(있는 경우)입니다. |
| `estimated_annual_revenue` | 비즈니스가 창출하는 추정 연간 매출입니다. |
| `num_employees` | 비즈니스에서 근무하는 직원 수입니다. |
| `review_count` | 비즈니스가 받은 총 리뷰 수입니다. |
| `reviews` | 비즈니스에 대한 고객 리뷰 및 피드백입니다. |
| `owner` | 비즈니스 소유자 또는 주요 담당자 이름입니다. |
| `manta_link` | Manta 플랫폼의 비즈니스 프로필 URL입니다. |
| `url` | 비즈니스와 연관된 일반 URL입니다. |
| `company_id` | 데이터셋 내 비즈니스의 고유 식별자입니다. |


## Use Cases

### 1. Market Analysis
데이터셋을 활용하여 시장 트렌드와 산업 벤치마크를 파악함으로써 전략 기획 및 의사결정을 지원합니다.

### 2. Business Development
비즈니스 상세 정보를 활용하여 잠재 파트너 또는 고객을 발굴하고, 네트워킹 및 성장 기회를 강화합니다.

### 3. Competitive Intelligence
경쟁사 정보를 분석하여 운영 방식과 시장 포지셔닝에 대한 인사이트를 확보하고, 경쟁 전략을 개선합니다.
